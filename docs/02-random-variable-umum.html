  7. menghubungkan ukuran-ukuran tersebut dengan pengambilan keputusan.

4.1 Pembuka

Pada bab sebelumnya, kita sudah melihat bahwa probabilitas menjadi penting ketika kita harus mengambil keputusan di bawah ketidakpastian. Kita juga sudah melihat bahwa salah satu cara paling kuat untuk “menjinakkan” ketidakpastian adalah dengan membuat model.

Di bab ini, kita masuk ke fondasi model tersebut: random variable.

Random variable adalah salah satu ide yang tampak sederhana, tetapi pengaruhnya sangat besar. Begitu kita berhasil memetakan hasil-hasil acak menjadi angka, kita memperoleh akses ke dunia matematika: kita bisa menggambar, menghitung, mensimulasikan, membandingkan, dan akhirnya memutuskan.

4.2 2.1 Quick Win: Dari Dunia Acak ke Garis Bilangan

Mari mulai dari contoh sederhana. Misalkan kita melempar dua koin. Ruang sampelnya adalah:

\[ S = \{HH, HT, TH, TT\} \]

Sekarang definisikan random variable:

\[ X = \text{jumlah sisi Head yang muncul} \]

Maka: - \(X(HH)=2\) - \(X(HT)=1\) - \(X(TH)=1\) - \(X(TT)=0\)

Artinya, hasil-hasil acak yang tadinya berupa simbol \(HH, HT, TH, TT\) dipetakan menjadi angka \(0,1,2\).

Mari lihat dengan Python.

import itertools
import numpy as np
import matplotlib.pyplot as plt

outcomes = list(itertools.product(['H', 'T'], repeat=2))

def X(outcome):
    return outcome.count('H')

mapped = [(o, X(o)) for o in outcomes]
mapped
[(('H', 'H'), 2), (('H', 'T'), 1), (('T', 'H'), 1), (('T', 'T'), 0)]
values = [X(o) for o in outcomes]
unique, counts = np.unique(values, return_counts=True)

plt.figure(figsize=(7,4))
plt.bar(unique, counts / counts.sum())
plt.xlabel("Nilai X = jumlah Head")
plt.ylabel("Probabilitas")
plt.title("Random variable sebagai pemetaan ke bilangan")
plt.xticks(unique)
plt.grid(alpha=0.3)
plt.show()

Dari sini kita melihat quick win yang penting: - event hidup di ruang sampel, - random variable memetakan event ke angka, - distribusi random variable memberi tahu kita bagaimana peluang tersebar pada angka-angka itu.

4.3 2.2 K — Konteks: Mengapa Kita Perlu Random Variable?

Bayangkan beberapa pertanyaan berikut: - Berapa jumlah kendaraan yang lewat gerbang tol dalam satu jam? - Berapa waktu tunggu pasien sampai dilayani? - Berapa profit harian sebuah klinik? - Berapa banyak produk cacat dalam 100 unit? - Berapa lama sebuah lampu bertahan sebelum rusak?

Semua pertanyaan ini punya satu pola: hasilnya tidak pasti, tetapi hasil itu berupa bilangan.

Itulah alasan random variable menjadi penting. Ia menjadi jembatan antara: - kejadian acak di dunia nyata, dan - analisis matematis berbasis angka.

4.4 2.3 M — Model: Definisi Random Variable

Secara formal, random variable \(X\) adalah sebuah fungsi yang memetakan setiap hasil \(\omega\) dalam ruang sampel \(S\) ke sebuah bilangan real:

\[ X: S \to \mathbb{R} \]

dengan

\[ \omega \mapsto X(\omega) \]

Di sini: - \(S\) = ruang sampel, - \(\omega\) = satu hasil acak, - \(X(\omega)\) = angka yang dihasilkan oleh pemetaan itu.

4.4.1 Catatan penting

Random variable bukan “angka yang acak” dalam arti longgar. Secara matematis, random variable adalah fungsi. Keacakan datang dari kenyataan bahwa hasil \(\omega\) belum diketahui sebelum eksperimen terjadi.

4.5 2.4 Event vs Nilai Random Variable

Mahasiswa sering bingung membedakan dua hal ini:

  • event = himpunan hasil dalam ruang sampel,
  • nilai random variable = angka hasil pemetaan.

Contoh: lempar dua dadu, lalu definisikan

\[ X = \text{jumlah mata dadu} \]

Maka event: \[ E = \{(1,6), (2,5), (3,4), (4,3), (5,2), (6,1)\} \]

adalah event “jumlah mata dadu = 7”.

Tetapi nilai random variable yang terkait adalah: \[ X = 7 \]

Jadi: - event adalah kumpulan outcome, - random variable adalah angka yang dihasilkan dari outcome itu.

Mari visualisasikan cepat.

outcomes = list(itertools.product(range(1, 7), repeat=2))
sums = [sum(o) for o in outcomes]

unique, counts = np.unique(sums, return_counts=True)
probs = counts / counts.sum()

plt.figure(figsize=(8,4))
plt.bar(unique, probs)
plt.xlabel("Nilai X = jumlah dua dadu")
plt.ylabel("Probabilitas")
plt.title("Distribusi random variable jumlah dua dadu")
plt.xticks(unique)
plt.grid(alpha=0.3)
plt.show()

4.6 2.5 Range: Nilai-Nilai yang Mungkin

Range dari random variable adalah himpunan semua nilai yang mungkin diambil oleh random variable tersebut.

Untuk contoh dua dadu dengan: \[ X = \text{jumlah mata dadu} \]

range-nya adalah: \[ \{2,3,4,\dots,12\} \]

Range ini penting karena memberi tahu: - apa saja kemungkinan nilai, - apakah nilai-nilai itu diskrit atau kontinu, - bagaimana bentuk distribusinya.

4.6.1 Diskrit vs Kontinu

4.6.1.1 Random Variable Diskrit

Jika range terdiri dari titik-titik terpisah, misalnya: \[ \{0,1,2,3,\dots\} \] maka random variable itu diskrit.

Contoh: - jumlah pelanggan datang, - jumlah cacat, - jumlah pesan masuk, - banyaknya Head.

4.6.1.2 Random Variable Kontinu

Jika range berupa interval pada garis bilangan, misalnya: \[ [0,\infty) \] atau \[ (-\infty,\infty) \] maka random variable itu kontinu.

Contoh: - waktu tunggu, - lama hidup komponen, - tinggi badan, - suhu.

4.7 2.6 PMF untuk Random Variable Diskrit

Jika \(X\) diskrit, maka distribusinya dapat dinyatakan dengan Probability Mass Function (PMF):

\[ p_X(x) = P(X=x) \]

PMF memberi peluang bahwa random variable mengambil nilai tepat \(x\).

4.7.1 Sifat PMF

  1. \[ p_X(x) \ge 0 \]

  2. \[ \sum_x p_X(x) = 1 \]

4.7.1.1 Contoh: jumlah Head pada dua lemparan koin

\[ P(X=0)=\frac{1}{4}, \quad P(X=1)=\frac{1}{2}, \quad P(X=2)=\frac{1}{4} \]

x = np.array([0, 1, 2])
pmf = np.array([1/4, 1/2, 1/4])

plt.figure(figsize=(7,4))
plt.stem(x, pmf, basefmt=" ")
plt.xlabel("x")
plt.ylabel("P(X=x)")
plt.title("PMF jumlah Head pada dua lemparan koin")
plt.grid(alpha=0.3)
plt.show()

4.7.1.2 Interpretasi keputusan

PMF membantu menjawab pertanyaan seperti: - seberapa mungkin tepat 2 cacat? - seberapa mungkin tepat 5 pelanggan datang? - seberapa mungkin tepat 1 dari 3 server gagal?

4.8 2.7 CDF: Bahasa Probabilitas Kumulatif

Baik untuk random variable diskrit maupun kontinu, salah satu fungsi terpenting adalah Cumulative Distribution Function (CDF):

\[ F_X(x) = P(X \le x) \]

CDF memberi peluang bahwa nilai random variable tidak melebihi \(x\).

Ini sangat penting karena banyak keputusan berbentuk: - peluang rusak sebelum waktu tertentu, - peluang demand tidak melebihi stok, - peluang profit lebih kecil dari nol, - peluang nilai ujian di bawah batas lulus.

4.8.0.1 Contoh CDF diskrit

Untuk \(X\) = jumlah Head pada dua lemparan koin: - \(F(0)=P(X\le 0)=1/4\) - \(F(1)=P(X\le 1)=3/4\) - \(F(2)=1\)

cdf = np.cumsum(pmf)

plt.figure(figsize=(7,4))
plt.step(x, cdf, where='post')
plt.xlabel("x")
plt.ylabel("F(x)=P(X≤x)")
plt.title("CDF random variable diskrit")
plt.ylim(-0.05, 1.05)
plt.grid(alpha=0.3)
plt.show()

4.9 2.8 PDF untuk Random Variable Kontinu

Untuk random variable kontinu, kita biasanya tidak berbicara tentang peluang tepat pada satu titik, karena:

\[ P(X=x)=0 \]

Sebagai gantinya, distribusi dijelaskan dengan Probability Density Function (PDF):

\[ f_X(x) \]

Peluang pada interval diperoleh melalui integral:

\[ P(a \le X \le b) = \int_a^b f_X(x)\,dx \]

4.9.1 Sifat PDF

  1. \[ f_X(x)\ge 0 \]

  2. \[ \int_{-\infty}^{\infty} f_X(x)\,dx = 1 \]

4.9.1.1 Contoh cepat: Uniform kontinu pada [0,1]

\[ f(x)= \begin{cases} 1, & 0 \le x \le 1\\ 0, & \text{lainnya} \end{cases} \]

Maka: \[ P(0.2 \le X \le 0.7)=0.5 \]

x = np.linspace(-0.2, 1.2, 400)
pdf = np.where((x >= 0) & (x <= 1), 1.0, 0.0)

plt.figure(figsize=(8,4))
plt.plot(x, pdf)
plt.fill_between(x, 0, pdf, where=((x >= 0.2) & (x <= 0.7)), alpha=0.3)
plt.xlabel("x")
plt.ylabel("f(x)")
plt.title("PDF Uniform(0,1)")
plt.grid(alpha=0.3)
plt.show()

4.9.1.2 Interpretasi penting

Tinggi PDF bukan peluang langsung.
Peluang adalah luas area di bawah kurva PDF.

4.10 2.9 Menghubungkan PMF, CDF, dan PDF

Hubungan penting:

4.10.1 Untuk diskrit

\[ F_X(x)=\sum_{t \le x} p_X(t) \]

4.10.2 Untuk kontinu

\[ F_X(x)=\int_{-\infty}^{x} f_X(t)\,dt \]

Jika PDF cukup halus, maka: \[ f_X(x)=\frac{d}{dx}F_X(x) \]

Jadi: - PMF/PDF memberi struktur lokal distribusi, - CDF memberi akumulasi peluang sampai titik tertentu.

4.11 2.10 Ekspektasi: Tebakan Terbaik Sebelum Fakta Terjadi

Ekspektasi adalah ukuran pusat distribusi. Ia sering dipandang sebagai rata-rata jangka panjang.

4.11.1 Untuk random variable diskrit

\[ E[X] = \sum_x x\,p_X(x) \]

4.11.2 Untuk random variable kontinu

\[ E[X] = \int_{-\infty}^{\infty} x f_X(x)\,dx \]

4.11.2.1 Contoh: jumlah Head pada dua lemparan koin

\[ E[X] = 0 \cdot \frac14 + 1 \cdot \frac12 + 2 \cdot \frac14 = 1 \]

Mari cek dengan simulasi.

rng = np.random.default_rng(123)

n = 100000
coin1 = rng.integers(0, 2, size=n)
coin2 = rng.integers(0, 2, size=n)
Xsim = coin1 + coin2

print("Rata-rata simulasi =", Xsim.mean())
Rata-rata simulasi = 1.00235

4.11.2.2 Interpretasi keputusan

Ekspektasi menjawab pertanyaan seperti: - rata-rata keuntungan harian, - rata-rata jumlah pelanggan, - rata-rata cacat per batch, - rata-rata waktu tunggu.

Tetapi ekspektasi tidak mengatakan seluruh cerita. Dua distribusi dengan mean sama bisa memiliki risiko yang sangat berbeda.

4.12 2.11 Varians dan Simpangan Baku: Seberapa Liar Penyimpangannya?

Varians mengukur seberapa jauh random variable menyebar dari nilai ekspektasinya.

4.12.1 Definisi

\[ \mathrm{Var}(X)=E[(X-E[X])^2] \]

Simpangan baku adalah akar kuadrat varians: \[ \sigma_X = \sqrt{\mathrm{Var}(X)} \]

4.12.2 Bentuk yang sering lebih praktis

\[ \mathrm{Var}(X)=E[X^2] - (E[X])^2 \]

4.12.2.1 Contoh: dua investasi dengan mean sama, variance beda

rng = np.random.default_rng(2026)

A = rng.choice([8, 10, 12], size=100000, p=[0.25, 0.50, 0.25])
B = rng.choice([0, 10, 20], size=100000, p=[0.25, 0.50, 0.25])

print("Mean A =", A.mean(), "Var A =", A.var(), "Std A =", A.std())
print("Mean B =", B.mean(), "Var B =", B.var(), "Std B =", B.std())
Mean A = 9.98924 Var A = 2.0013242224 Std A = 1.414681668220805
Mean B = 9.9584 Var B = 49.78426944 Std B = 7.055796867824356
plt.figure(figsize=(8,4))
plt.hist(A, bins=np.arange(-0.5, 21.5, 1), alpha=0.6, density=True, label="A")
plt.hist(B, bins=np.arange(-0.5, 21.5, 1), alpha=0.6, density=True, label="B")
plt.xlabel("Nilai")
plt.ylabel("Density / Probability")
plt.title("Mean sama, variance berbeda")
plt.legend()
plt.grid(alpha=0.3)
plt.show()

4.12.2.2 Interpretasi keputusan

Jika kita hanya melihat mean, dua pilihan ini tampak setara. Tetapi jika kita juga melihat variance, kita sadar bahwa salah satu pilihan jauh lebih berisiko.

4.13 2.12 Sifat-Sifat Penting Ekspektasi dan Varians

4.13.1 Linearitas ekspektasi

Untuk konstanta \(a,b\): \[ E[aX+b] = aE[X]+b \]

Ini sangat penting karena memungkinkan kita memindahkan model ke unit baru atau mengubah skala.

4.13.2 Varians terhadap transformasi linear

\[ \mathrm{Var}(aX+b)=a^2 \mathrm{Var}(X) \]

Artinya: - menambah konstanta tidak mengubah variance, - mengalikan skala dengan \(a\) memperbesar variance sebesar \(a^2\).

Mari cek.

rng = np.random.default_rng(0)
X = rng.normal(loc=5, scale=2, size=100000)

Y = 3*X + 10

print("E[X] ~", X.mean())
print("E[Y] ~", Y.mean(), "   teori =", 3*X.mean() + 10)

print("Var[X] ~", X.var())
print("Var[Y] ~", Y.var(), "   teori =", 9*X.var())
E[X] ~ 4.998183498453758
E[Y] ~ 24.994550495361278    teori = 24.994550495361274
Var[X] ~ 4.001028098478324
Var[Y] ~ 36.00925288630492    teori = 36.009252886304914

4.14 2.13 Diskrit dan Kontinu dalam Praktik

Secara praktik, banyak data diukur secara diskrit: - berat badan dibulatkan ke 0.1 kg, - waktu dicatat dalam detik, - panjang dicatat dalam milimeter.

Tetapi secara model, kadang jauh lebih sederhana menganggap besaran itu kontinu. Ini bukan kesalahan, tetapi bentuk idealisasi yang membantu analisis.

4.14.1 Pedoman intuitif

  • Jika nilai alami berupa hitungan → cenderung diskrit
  • Jika nilai alami berupa pengukuran halus → sering dimodelkan kontinu

Contoh: - jumlah mobil → diskrit - waktu hidup lampu → kontinu - jumlah pasien → diskrit - suhu → kontinu

4.15 2.14 Dari Histogram ke Distribusi

Dalam kenyataan, kita sering tidak mulai dari rumus distribusi, tetapi dari data. Dari data, kita bisa membuat histogram untuk melihat bentuk penyebaran.

Mari contohkan data simulasi.

rng = np.random.default_rng(99)
samples = rng.normal(loc=100, scale=15, size=5000)

plt.figure(figsize=(8,4))
plt.hist(samples, bins=40, density=True, alpha=0.75, edgecolor='black')
plt.xlabel("Nilai")
plt.ylabel("Density")
plt.title("Histogram sebagai pendekatan bentuk distribusi")
plt.grid(alpha=0.3)
plt.show()

Histogram tidak otomatis menjadi PMF atau PDF yang eksak, tetapi histogram sangat berguna untuk: - menduga bentuk distribusi, - melihat pusat, - melihat penyebaran, - mendeteksi skewness atau outlier, - memilih model awal yang masuk akal.

4.16 2.15 KMQA Mini-Case 1 — Timbangan Berat Badan

4.16.1 K — Konteks

Sebuah timbangan digital menampilkan berat badan dalam satu angka desimal.

4.16.2 M — Model

Jika \(X\) adalah berat badan orang yang naik ke timbangan, maka \(X\) dapat dimodelkan sebagai random variable kontinu, walaupun tampilan akhirnya dibulatkan.

4.16.3 Q — Questions

  1. Apa range masuk akalnya?
  2. Apakah lebih masuk akal dipandang diskrit atau kontinu?
  3. Ukuran apa yang penting: mean, variance, atau keduanya?

4.16.4 A — Apply

Jawabannya: - range teknis mungkin 0–999.9, tetapi range realistis jauh lebih sempit, - secara teori lebih nyaman dipandang kontinu, - mean memberi pusat berat populasi, - variance memberi seberapa beragam berat badan orang yang diukur.

4.17 2.16 KMQA Mini-Case 2 — Jumlah Kendaraan di Gerbang Tol

4.17.1 K — Konteks

Kita ingin memodelkan jumlah kendaraan yang lewat dalam satu jam.

4.17.2 M — Model

Jika \(X\) = jumlah kendaraan, maka \(X\) adalah random variable diskrit.

4.17.3 Q — Questions

  1. Mengapa diskrit?
  2. Apa range yang masuk akal?
  3. Apa arti \(P(X=120)\)? Apa arti \(P(X\le 120)\)?

4.17.4 A — Apply

  • diskrit karena berupa hitungan,
  • \(P(X=120)\) adalah peluang tepat 120 kendaraan,
  • \(P(X\le 120)\) adalah peluang jumlah kendaraan tidak melebihi 120.

4.18 2.17 KMQA Mini-Case 3 — Waktu Tunggu Pasien

4.18.1 K — Konteks

Sebuah klinik ingin memodelkan waktu tunggu pasien hingga dilayani.

4.18.2 M — Model

Jika \(X\) = waktu tunggu (menit), maka \(X\) biasanya dimodelkan kontinu.

4.18.3 Q — Questions

  1. Mengapa kontinu?
  2. Mengapa \(P(X=10)=0\) tetapi \(P(9.5 \le X \le 10.5)\) bisa positif?
  3. Mengapa CDF penting dalam konteks ini?

4.18.4 A — Apply

  • kontinu karena waktu dapat berubah halus,
  • pada model kontinu, peluang titik tunggal nol,
  • yang bermakna adalah peluang interval,
  • CDF membantu menjawab “peluang menunggu tidak lebih dari x menit”.

from scipy import stats

4.19.2 PMF / PDF / CDF dari SciPy

x_discrete = np.arange(0, 11)
pmf_binom = stats.binom.pmf(x_discrete, n=10, p=0.4)

x_cont = np.linspace(-3, 3, 200)
pdf_norm = stats.norm.pdf(x_cont, loc=0, scale=1)

4.25.1 Prinsip-Prinsip Dasar yang Digunakan

Berdasarkan materi kuliah Random Variable, berikut adalah prinsip utama untuk menyelesaikan Problem Set 3: 1. Random Variable Diskrit: Memiliki himpunan nilai yang terhitung. Fungsi Probabilitas (PMF) dinotasikan dengan \(f(x) = P(X = x)\) dengan syarat \(\sum f(x) = 1\). 2. Random Variable Kontinu: Nilainya terdefinisi pada interval tak terhitung, dan probabilitas adalah luas area di bawah kurva Fungsi Kepadatan Probabilitas (PDF), dihitung dengan integral \(P(a < X < b) = \int_a^b f(x) dx\). 3. Fungsi Distribusi Kumulatif (CDF): CDF atau \(F(x)\) adalah akumulasi probabilitas \(P(X \le x)\). Pada fungsi kontinu, \(f(x)\) adalah turunan pertama dari CDF, yaitu \(f(x) = dF(x)/dx\). 4. Nilai Harap (Mean/Ekspektasi) dan Variansi: Mean (\(\mu\)) menunjukkan pusat distribusi probabilitas, dihitung dengan \(\sum x f(x)\) untuk diskrit dan \(\int x f(x) dx\) untuk kontinu. Variansi (\(\sigma^2\)) mengukur penyebaran, dihitung dengan \(E(X^2) - \mu^2\).

4.25.2 Solusi Bebrapa Soal Problem Set 3

Soal 1: Analisis Distribusi Diskrit * Masalah: Diketahui \(X\) diskrit dengan \(P(0,2)=0,1; P(0,4)=0,2; P(0,5)=0,2; P(0,8)=0,3; P(1)=0,2\). * Solusi: * Range (\(R_x\)): \(\{0,2;\; 0,4;\; 0,5;\; 0,8;\; 1\}\). * \(P(X \le 0,5)\): \(0,1 + 0,2 + 0,2 = \mathbf{0,5}\). * \(P(0,25 < X < 0,75)\): Nilai yang masuk hanyalah 0,4 dan 0,5. Jumlahnya \(= 0,2 + 0,2 = \mathbf{0,4}\) (Catatan: Pada dokumen sumber terdapat ralat perhitungan manual di mana probabilitas \(0,8\) ikut ditambahkan sehingga menjadi \(0,7\)). * Probabilitas Bersyarat \(P(X=0,2 \mid X<0,6)\): \(P(X=0,2) / P(X<0,6) = 0,1 / (0,1+0,2+0,2) = 0,1 / 0,5 = \mathbf{0,2}\). * Mean & Variansi: \(\mu = \sum xP(x) = \mathbf{0,64}\). Variansi \(\sigma^2 = E(X^2) - \mu^2 = 0,478 - 0,64^2 = \mathbf{0,0684}\).

Soal 2: Fungsi Distribusi Kumulatif (CDF) Eksponensial Khusus * Masalah: \(F(x) = 1 - e^{-x^2}\) untuk \(x > 0\). Berapa probabilitas \(X > 1\)?. * Solusi: \(P(X > 1) = 1 - P(X \le 1) = 1 - F(1) = 1 - (1 - e^{-1}) = e^{-1} \approx \mathbf{0,3679}\).

Soal 3 & 4: Pemodelan Fungsi Massa Probabilitas (PMF) * Masalah: Menerjemahkan probabilitas kesuksesan produk menjadi fungsi PMF pendapatan \(X\). * Solusi: * Soal 3: \(f(10M) = 0,1;\; f(50M) = 0,6;\; f(100M) = 0,3\). * Soal 4: \(f(100M) = 0,2;\; f(250M) = 0,3;\; f(500M) = 0,5\).

Soal 5: PDF Berupa Garis Linear (\(f(x) = kx\)) * Masalah: \(f(x) = kx\) untuk \(0 \le x \le 4\). Tentukan konstanta \(k\), CDF, dan probabilitas pada rentang tertentu. * Solusi: * Mencari \(k\): \(\int_0^4 kx dx = 1 \implies 8k = 1 \implies \mathbf{k = 1/8}\). * Fungsi CDF (\(F(x)\)): \(\int_0^x \frac{1}{8}t dt = \mathbf{x^2/16}\). * \(P(X < 1/2)\): \(F(1/2) = (1/2)^2 / 16 = \mathbf{1/64}\). * \(P(1/2 < X < 1)\): \(F(1) - F(1/2) = \frac{1}{16} - \frac{1}{64} = \mathbf{3/64}\).

Soal 6: Distribusi Pareto (Waktu Komputasi Algoritma) * Masalah: \(F(r) = 1 - (1/r)^\alpha\). Diketahui \(P(R > 10) = 0,5\). Cari \(P(R > 1000)\). * Solusi: Substitusi \(P(R \le 10) = 0,5 \implies 1 - (1/10)^\alpha = 0,5 \implies 10^{-\alpha} = 0,5 \implies \alpha = -\log(0,5)\). Lalu hitung target: \(P(R > 1000) = (1/1000)^{-\log(0,5)} = \mathbf{0,125}\).

Soal 7: Distribusi Seragam Senyawa Kimia * Masalah: \(f(x) = 2,0\) pada interval \(49,75 < x < 50,25\). Jika \(Y\) melambangkan \(90\%\) dari \(X\), tentukan nilai batas intervalnya. * Solusi: \(P(X > 50) = \int_{50}^{50,25} 2,0 dx = \mathbf{0,5}\). Transformasi \(Y = 0,9X \implies X = Y/0,9 \approx 1,111Y\). Maka interval batas baru \(Y\) adalah \(49,75 < 1,111Y < 50,25 \implies \mathbf{44,775 < y < 47,25}\).

Soal 8: Ekstraksi PDF dari CDF Terpotong (Piecewise) * Masalah: Diketahui CDF \(F(x) = 0,2x\) (untuk \(0 \le x < 4\)) dan \(0,04x + 0,64\) (untuk \(4 \le x < 9\)). Hitung Ekspektasi \(E(X)\) dan \(P(X \ge 6)\). * Solusi: * Turunkan \(F(x)\) untuk mendapat \(f(x)\): Diperoleh \(f(x) = 0,2\) pada \(x \in [0,4)\) dan \(f(x) = 0,04\) pada \(x \in [4,9)\). * Ekspektasi \(E(X)\): \(\int_0^4 x(0,2) dx + \int_4^9 x(0,04) dx = 1,6 + 1,3 = \mathbf{2,9}\). * \(P(X \ge 6)\): \(1 - F(6) = 1 - (0,04(6) + 0,64) = 1 - 0,88 = \mathbf{0,12}\).

4.25.3 Desain Kode Python (Class ProbSet3Solver)

Kode berikut menyediakan alat matematis berbasis fungsi integrasi scipy dan numpy untuk menyelesaikan seluruh logika soal di atas secara reusable.

import numpy as np
import scipy.integrate as integrate
import math

class ProbSet3Solver:
    """Class penyelesaian Problem Set 3 Probabilitas & Statistik"""
    
    @staticmethod
    def discrete_stats(x_vals, p_vals):
        """Menghitung Mean dan Variansi Distribusi Diskrit (Soal 1)"""
        mean = sum(x * p for x, p in zip(x_vals, p_vals))
        e_x_square = sum((x**2) * p for x, p in zip(x_vals, p_vals))
        variance = e_x_square - (mean**2)
        return mean, variance

    @staticmethod
    def discrete_conditional(x_vals, p_vals, target_x, condition_max):
        """P(X = target_x | X < condition_max)"""
        p_target = p_vals[x_vals.index(target_x)]
        p_condition = sum(p for x, p in zip(x_vals, p_vals) if x < condition_max)
        return p_target / p_condition

    @staticmethod
    def continuous_cdf_exp_square(x):
        """Evaluasi F(x) = 1 - e^(-x^2) untuk mencari P(X > x) (Soal 2)"""
        f_x = 1 - math.exp(-(x**2))
        return 1 - f_x # P(X > x)

    @staticmethod
    def find_pdf_constant(func_template, a, b):
        """Mencari konstanta k agar Integral f(x) = 1 (Soal 5)"""
        # func_template adalah fungsi tanpa k, misal x -> x
        integral_val, _ = integrate.quad(func_template, a, b)
        k = 1 / integral_val
        return k

    @staticmethod
    def pareto_probability(r_known, p_known_greater, r_target):
        """Menyelesaikan Soal Pareto dengan mencari alpha dari logaritma (Soal 6)"""
        # P(R > r_known) = p_known_greater  => (1/r_known)^alpha = p_known_greater
        # alpha * log(1/r_known) = log(p_known_greater)
        alpha = math.log(p_known_greater) / math.log(1 / r_known)
        # Menghitung probabilitas target P(R > r_target)
        return (1 / r_target)**alpha

    @staticmethod
    def piecewise_expected_value(funcs, bounds):
        """
        Mencari E(X) dari fungsi terpotong (Piecewise) (Soal 8).
        funcs = [f1(x), f2(x)], bounds = [(a,b), (c,d)]
        """
        expected_value = 0
        for f, (a, b) in zip(funcs, bounds):
            val, _ = integrate.quad(lambda x: x * f(x), a, b)
            expected_value += val
        return expected_value
