  2. membangun distribusi kontinu sederhana dari tabel selang atau pendekatan trapezoid,
  3. mengenali dan menggunakan distribusi kontinu penting:
    • uniform kontinu,

  4. memahami hubungan antardistribusi,

6.1 Pembuka

Di bab sebelumnya, kita banyak berurusan dengan besaran yang berupa hitungan: - jumlah cacat, - jumlah pelanggan, - jumlah sukses, - jumlah percobaan.

Sekarang kita beralih ke dunia yang sedikit berbeda: dunia pengukuran.

Kita akan memodelkan besaran seperti: - waktu tunggu, - umur hidup lampu, - tinggi badan, - error pengukuran, - besar permintaan, - lama layanan, - dan waktu antar kedatangan.

Pada banyak situasi seperti ini, lebih masuk akal memodelkan nilai sebagai kontinu. Nilainya tidak sekadar 0, 1, 2, 3, tetapi berada pada rentang yang halus di garis bilangan.

6.2 4.1 Quick Win: Mengapa PDF Bukan Peluang Langsung?

Mari ambil contoh yang sederhana. Misalkan \(X\) adalah random variable kontinu yang uniform pada interval [0, 1]. Kita tahu PDF-nya adalah:

\[ f(x)=1 \quad \text{untuk } 0 \le x \le 1 \]

Sekarang pertanyaannya: - berapa \(P(X=0.5)\)? - berapa \(P(0.4 \le X \le 0.6)\)?

import numpy as np
import matplotlib.pyplot as plt
from scipy import stats

## Uniform(0,1)
print("P(X=0.5) untuk model kontinu = 0")
print("P(0.4 <= X <= 0.6) =", 0.6 - 0.4)
P(X=0.5) untuk model kontinu = 0
P(0.4 <= X <= 0.6) = 0.19999999999999996
x = np.linspace(-0.2, 1.2, 400)
pdf = np.where((x >= 0) & (x <= 1), 1.0, 0.0)

plt.figure(figsize=(8,4))
plt.plot(x, pdf)
plt.fill_between(x, 0, pdf, where=((x >= 0.4) & (x <= 0.6)), alpha=0.3)
plt.xlabel("x")
plt.ylabel("f(x)")
plt.title("Pada RV kontinu, peluang = luas area, bukan tinggi kurva")
plt.grid(alpha=0.3)
plt.show()

Dari quick win ini kita memperoleh satu pelajaran besar:

Untuk random variable kontinu, peluang tidak datang dari tinggi titik, tetapi dari luas area di bawah kurva PDF.

6.3 4.2 Distribusi Kontinu Kustom dari Tabel Selang / Trapezoid

6.3.1 K — Konteks

Kadang kita tidak mulai dari distribusi terkenal seperti Normal atau Exponential. Kita hanya punya perkiraan bentuk dari domain expert atau dari tabel interval. Misalnya: - permintaan produk kemungkinan besar di tengah, menurun di tepi, - usia layanan komponen diperkirakan lebih padat pada interval tertentu, - waktu tempuh berada dalam rentang tertentu dengan perubahan kerapatan yang kasar.

6.3.2 M — Model

Kita dapat membentuk PDF kustom dari titik-titik \((x_i, f_i)\), lalu menggunakan interpolasi linier atau pendekatan trapezoid. Selama luas totalnya dinormalisasi menjadi 1, fungsi itu dapat dipakai sebagai PDF.

6.3.3 Q — Questions

  1. Bagaimana membangun PDF kontinu dari tabel?
  2. Bagaimana memastikan luas total = 1?
  3. Bagaimana menghitung peluang interval secara numerik?

6.3.4 A — Apply

## Titik-titik pembentuk "density" mentah
x_pts = np.array([0, 2, 4, 6, 8, 10], dtype=float)
y_raw = np.array([0.0, 0.1, 0.25, 0.2, 0.08, 0.0], dtype=float)

## Normalisasi agar integral = 1
area_raw = np.trapezoid(y_raw, x_pts)
y = y_raw / area_raw

x_grid = np.linspace(0, 10, 500)
y_grid = np.interp(x_grid, x_pts, y)
plt.figure(figsize=(8,4))
plt.plot(x_grid, y_grid, label="PDF kustom (interpolasi linier)")
plt.scatter(x_pts, y, color='red')
plt.xlabel("x")
plt.ylabel("f(x)")
plt.title("PDF kontinu kustom dari tabel selang")
plt.legend()
plt.grid(alpha=0.3)
plt.show()
## Peluang interval [3, 7]
mask = (x_grid >= 3) & (x_grid <= 7)
prob_3_7 = np.trapezoid(y_grid[mask], x_grid[mask])
print("P(3 <= X <= 7) ~", prob_3_7)

## Mean numerik
mean_num = np.trapezoid(x_grid * y_grid, x_grid)
var_num = np.trapezoid((x_grid - mean_num)**2 * y_grid, x_grid)
print("Mean ~", mean_num)
print("Variance ~", var_num)
P(3 <= X <= 7) ~ 0.659204179902892
Mean ~ 4.825378851610921
Variance ~ 3.92183354068992

6.3.4.1 Interpretasi

Distribusi kontinu kustom penting ketika: - data atau pengetahuan domain belum cukup untuk memilih distribusi tertutup tertentu, - kita tetap ingin menghitung peluang, mean, variance, dan quantile, - pendekatan numerik lebih realistis daripada memaksakan model yang belum tentu cocok.

6.4 4.3 Uniform Kontinu

6.4.1 K — Konteks

Distribusi uniform kontinu cocok ketika nilai dianggap tersebar merata pada sebuah interval. Misalnya: - waktu kedatangan acak di antara dua titik waktu, - lokasi acak pada segmen garis, - nilai acak yang dipilih secara merata dari suatu rentang.

6.4.2 M — Model

Jika: \[ X \sim \mathrm{Uniform}(a,b) \] maka PDF-nya:

\[ f(x)=\frac{1}{b-a}, \qquad a \le x \le b \]

dan nol di luar interval tersebut.

6.4.3 Q — Questions

  1. Berapa peluang \(X\) berada pada sub-interval tertentu?
  2. Berapa mean dan variance?
  3. Kapan asumsi “semua nilai di interval sama padat” masuk akal?

6.4.4 A — Apply

a, b = 2, 8
x = np.linspace(0, 10, 400)
pdf = np.where((x >= a) & (x <= b), 1/(b-a), 0.0)

plt.figure(figsize=(8,4))
plt.plot(x, pdf)
plt.fill_between(x, 0, pdf, where=((x >= 3) & (x <= 5)), alpha=0.3)
plt.xlabel("x")
plt.ylabel("f(x)")
plt.title("PDF Uniform(2,8)")
plt.grid(alpha=0.3)
plt.show()
print("P(3 <= X <= 5) =", (5-3)/(8-2))
print("Mean =", (a+b)/2)
print("Variance =", (b-a)**2 / 12)
P(3 <= X <= 5) = 0.3333333333333333
Mean = 5.0
Variance = 3.0

6.4.4.1 Hasil teoritis

\[ E[X]=\frac{a+b}{2} \]

\[ \mathrm{Var}(X)=\frac{(b-a)^2}{12} \]

6.4.5 Latihan singkat

  1. Jika \(X \sim \mathrm{Uniform}(10,20)\), hitung \(P(12 \le X \le 15)\).
  2. Hitung mean dan variance-nya.
  3. Beri satu contoh aplikasi nyata yang cocok dimodelkan uniform kontinu.

6.5 4.4 Normal Distribution

6.5.1 K — Konteks

Normal adalah salah satu distribusi paling terkenal karena sering muncul pada: - error pengukuran, - tinggi badan, - berat badan, - hasil agregasi banyak pengaruh kecil, - noise sensor.

6.5.2 M — Model

Jika: \[ X \sim \mathcal{N}(\mu, \sigma^2) \] maka PDF-nya:

\[ f(x)=\frac{1}{\sigma \sqrt{2\pi}}\exp\left(-\frac{(x-\mu)^2}{2\sigma^2}\right) \]

Parameter: - \(\mu\): pusat distribusi, - \(\sigma\): skala penyebaran.

6.5.3 Q — Questions

  1. Berapa peluang \(X\) kurang dari nilai tertentu?
  2. Berapa peluang \(X\) berada dalam interval tertentu?
  3. Bagaimana pengaruh \(\mu\) dan \(\sigma\) pada bentuk kurva?

6.5.4 A — Apply

mu, sigma = 100, 15
x = np.linspace(40, 160, 500)
pdf = stats.norm.pdf(x, loc=mu, scale=sigma)

plt.figure(figsize=(8,4))
plt.plot(x, pdf)
plt.xlabel("x")
plt.ylabel("f(x)")
plt.title("PDF Normal(100, 15^2)")
plt.grid(alpha=0.3)
plt.show()
print("P(X < 90) =", stats.norm.cdf(90, loc=mu, scale=sigma))
print("P(90 <= X <= 110) =", stats.norm.cdf(110, loc=mu, scale=sigma) - stats.norm.cdf(90, loc=mu, scale=sigma))
P(X < 90) = 0.2524925375469229
P(90 <= X <= 110) = 0.4950149249061542

6.5.4.1 Hubungan dengan Binomial

Jika: - \(X \sim \mathrm{Binomial}(n,p)\), - \(n\) besar,

maka Binomial dapat didekati oleh Normal dengan:

\[ \mu = np, \qquad \sigma^2 = np(1-p) \]

Mari lihat contoh.

n, p = 100, 0.4
mu_bin = n*p
sigma_bin = np.sqrt(n*p*(1-p))

x = np.arange(20, 61)
pmf_bin = stats.binom.pmf(x, n=n, p=p)

x_cont = np.linspace(20, 60, 500)
pdf_norm = stats.norm.pdf(x_cont, loc=mu_bin, scale=sigma_bin)

plt.figure(figsize=(8,4))
plt.stem(x, pmf_bin, basefmt=" ", label="Binomial")
plt.plot(x_cont, pdf_norm, 'r', label="Normal approximation")
plt.xlabel("x")
plt.ylabel("Probability / Density")
plt.title("Pendekatan Normal terhadap Binomial")
plt.legend()
plt.grid(alpha=0.3)
plt.show()

6.5.4.2 Interpretasi keputusan

Normal sering dipakai untuk: - menetapkan toleransi manufaktur, - menghitung peluang deviasi pengukuran, - mengevaluasi performa relatif terhadap target.

6.5.5 Latihan singkat

  1. Jika \(X \sim \mathcal{N}(70,10^2)\), hitung \(P(X < 60)\).
  2. Hitung \(P(65 \le X \le 80)\).
  3. Simulasikan 10000 sampel dari distribusi ini dan bandingkan histogram dengan PDF.

6.6 4.5 Gamma Distribution

6.6.1 K — Konteks

Distribusi Gamma sangat berguna untuk memodelkan: - waktu tunggu sampai beberapa kejadian terjadi, - lifetime positif dengan bentuk fleksibel, - akumulasi beberapa waktu tunggu eksponensial.

6.6.2 M — Model

Salah satu parametrisasinya: \[ X \sim \mathrm{Gamma}(\alpha, \theta) \]

dengan: - \(\alpha\) = shape, - \(\theta\) = scale.

PDF-nya:

\[ f(x)=\frac{1}{\Gamma(\alpha)\theta^\alpha}x^{\alpha-1}e^{-x/\theta}, \qquad x>0 \]

6.6.3 Q — Questions

  1. Apa pengaruh shape dan scale?
  2. Kapan Gamma lebih cocok daripada Exponential?
  3. Bagaimana menghitung peluang waktu selesai sebelum batas tertentu?

6.6.4 A — Apply

alpha, theta = 3, 2
x = np.linspace(0, 30, 500)
pdf = stats.gamma.pdf(x, a=alpha, scale=theta)

plt.figure(figsize=(8,4))
plt.plot(x, pdf)
plt.xlabel("x")
plt.ylabel("f(x)")
plt.title("PDF Gamma(shape=3, scale=2)")
plt.grid(alpha=0.3)
plt.show()
print("Mean =", alpha * theta)
print("Variance =", alpha * theta**2)
print("P(X <= 8) =", stats.gamma.cdf(8, a=alpha, scale=theta))
Mean = 6
Variance = 12
P(X <= 8) = 0.7618966944464556

6.6.4.1 Hasil teoritis

\[ E[X]=\alpha\theta \]

\[ \mathrm{Var}(X)=\alpha\theta^2 \]

6.6.5 Interpretasi

Gamma cocok untuk waktu positif yang tidak memoryless dan memiliki bentuk yang bisa semakin “membukit” saat shape membesar.

6.7 4.6 Exponential Distribution

6.7.1 K — Konteks

Distribusi Exponential sangat terkenal untuk memodelkan: - waktu antar kedatangan pada Poisson process, - waktu sampai kegagalan pertama pada sistem memoryless, - waktu tunggu hingga kejadian berikutnya.

6.7.2 M — Model

Jika: \[ X \sim \mathrm{Exponential}(\lambda) \] maka PDF-nya:

\[ f(x)=\lambda e^{-\lambda x}, \qquad x \ge 0 \]

CDF-nya:

\[ F(x)=1-e^{-\lambda x} \]

6.7.3 Q — Questions

  1. Berapa peluang menunggu kurang dari \(t\)?
  2. Mengapa distribusi ini memoryless?
  3. Bagaimana hubungannya dengan Poisson process?

6.7.4 A — Apply

lam = 0.5
x = np.linspace(0, 15, 500)
pdf = stats.expon.pdf(x, scale=1/lam)

plt.figure(figsize=(8,4))
plt.plot(x, pdf)
plt.xlabel("x")
plt.ylabel("f(x)")
plt.title("PDF Exponential(rate=0.5)")
plt.grid(alpha=0.3)
plt.show()
print("Mean =", 1/lam)
print("Variance =", 1/lam**2)
print("P(X <= 3) =", stats.expon.cdf(3, scale=1/lam))
print("P(X > 5) =", 1 - stats.expon.cdf(5, scale=1/lam))
Mean = 2.0
Variance = 4.0
P(X <= 3) = 0.7768698398515702
P(X > 5) = 0.08208499862389884

6.7.4.1 Sifat memoryless

\[ P(X>s+t \mid X>s)=P(X>t) \]

Distribusi Exponential adalah versi kontinu dari ide memoryless yang di dunia diskrit dimiliki distribusi Geometric.

6.7.4.2 Hubungan dengan Poisson process

Jika jumlah kejadian per interval mengikuti Poisson process dengan rate \(\lambda\), maka waktu antar kedatangan mengikuti Exponential(\(\lambda\)).

Mari simulasikan.

rng = np.random.default_rng(123)
lam = 2.0  ## rata-rata 2 kejadian per satuan waktu

interarrival = rng.exponential(scale=1/lam, size=50000)

x = np.linspace(0, np.percentile(interarrival, 99.5), 400)
pdf = stats.expon.pdf(x, scale=1/lam)

plt.figure(figsize=(8,4))
plt.hist(interarrival, bins=60, density=True, alpha=0.6, label="Simulasi")
plt.plot(x, pdf, label="PDF teoritis")
plt.xlabel("Interarrival time")
plt.ylabel("Density")
plt.title("Interarrival time pada Poisson process ~ Exponential")
plt.legend()
plt.grid(alpha=0.3)
plt.show()

6.7.5 Latihan singkat

  1. Jika rata-rata waktu tunggu 10 menit, berapa rate \(\lambda\)?
  2. Hitung peluang waktu tunggu kurang dari 5 menit.
  3. Verifikasi sifat memoryless dengan simulasi.

6.8 4.7 Erlang Distribution

6.8.1 K — Konteks

Erlang muncul ketika kita menjumlahkan beberapa waktu tunggu Eksponensial independen dengan rate yang sama. Misalnya: - total waktu layanan yang melewati beberapa tahap berurutan, - waktu sampai \(k\) kejadian dalam Poisson process.

6.8.2 M — Model

Jika: \[ X = X_1 + X_2 + \cdots + X_k \] dengan tiap \(X_i \sim \text{Exponential}(\lambda)\) independen, maka:

\[ X \sim \text{Erlang}(k,\lambda) \]

Erlang adalah kasus khusus Gamma dengan shape integer.

6.8.3 Q — Questions

  1. Bagaimana bentuk distribusinya dibanding Exponential?
  2. Bagaimana pengaruh \(k\)?
  3. Apa arti praktis “jumlah beberapa tahap layanan”?

6.8.4 A — Apply

lam = 0.5
k = 3

x = np.linspace(0, 25, 500)
pdf = stats.gamma.pdf(x, a=k, scale=1/lam)

plt.figure(figsize=(8,4))
plt.plot(x, pdf)
plt.xlabel("x")
plt.ylabel("f(x)")
plt.title("PDF Erlang(k=3, rate=0.5)")
plt.grid(alpha=0.3)
plt.show()
print("Mean =", k/lam)
print("Variance =", k/(lam**2))
print("P(X <= 8) =", stats.gamma.cdf(8, a=k, scale=1/lam))
Mean = 6.0
Variance = 12.0
P(X <= 8) = 0.7618966944464556

6.8.4.1 Interpretasi

Jika suatu proses terdiri dari 3 tahap memoryless independen, maka total waktunya tidak lagi Exponential, tetapi Erlang-3.

6.9 4.8 Weibull Distribution

6.9.1 K — Konteks

Weibull sangat populer dalam reliability engineering karena dapat memodelkan berbagai pola kegagalan: - kegagalan awal, - kegagalan acak, - kegagalan akibat keausan.

6.9.2 M — Model

Jika: \[ X \sim \text{Weibull}(k,\lambda) \] dengan: - \(k\) = shape, - \(\lambda\) = scale,

maka PDF-nya adalah:

\[ f(x)=\frac{k}{\lambda}\left(\frac{x}{\lambda}\right)^{k-1}e^{-(x/\lambda)^k}, \qquad x \ge 0 \]

6.9.3 Q — Questions

  1. Bagaimana shape \(k\) memengaruhi hazard?
  2. Kapan Weibull lebih realistis daripada Exponential?
  3. Bagaimana peluang komponen gagal sebelum waktu tertentu?

6.9.4 A — Apply

k_shape = 2.0
lam_scale = 10.0

x = np.linspace(0, 30, 500)
pdf = stats.weibull_min.pdf(x, c=k_shape, scale=lam_scale)

plt.figure(figsize=(8,4))
plt.plot(x, pdf)
plt.xlabel("x")
plt.ylabel("f(x)")
plt.title("PDF Weibull(shape=2, scale=10)")
plt.grid(alpha=0.3)
plt.show()
print("P(X <= 8) =", stats.weibull_min.cdf(8, c=k_shape, scale=lam_scale))
P(X <= 8) = 0.47270757595695145

6.9.4.1 Interpretasi

Weibull lebih fleksibel daripada Exponential. Ia sangat berguna saat laju kegagalan tidak konstan.

6.10 4.9 Pareto Distribution

6.10.1 K — Konteks

Pareto dipakai untuk fenomena heavy-tail, misalnya: - kekayaan, - ukuran file, - kerugian ekstrem, - pendapatan segelintir pihak yang sangat besar.

6.10.2 M — Model

Jika: \[ X \sim \text{Pareto}(x_m,\alpha) \] maka untuk \(x \ge x_m\):

\[ f(x)=\alpha \frac{x_m^\alpha}{x^{\alpha+1}} \]

6.10.3 Q — Questions

  1. Mengapa Pareto menghasilkan peluang ekstrem yang lebih besar?
  2. Kapan model heavy-tail penting dalam keputusan?
  3. Mengapa mean/variance bisa bermasalah untuk parameter tertentu?

6.10.4 A — Apply

alpha = 3.0
xm = 1.0

x = np.linspace(1, 10, 500)
pdf = stats.pareto.pdf(x, b=alpha, scale=xm)

plt.figure(figsize=(8,4))
plt.plot(x, pdf)
plt.xlabel("x")
plt.ylabel("f(x)")
plt.title("PDF Pareto")
plt.grid(alpha=0.3)
plt.show()
print("P(X > 3) =", 1 - stats.pareto.cdf(3, b=alpha, scale=xm))
P(X > 3) = 0.03703703703703698

6.10.4.1 Interpretasi

6.11 4.10 Chi-Square Distribution

6.11.1 K — Konteks

Chi-Square muncul dalam: - inferensi statistik, - goodness-of-fit, - analisis varians, - penjumlahan kuadrat peubah normal baku.

6.11.2 M — Model

Jika: \[ Z_1, Z_2, \dots, Z_k \sim \mathcal{N}(0,1) \] independen, maka:

\[ X = Z_1^2 + Z_2^2 + \cdots + Z_k^2 \sim \chi^2(k) \]

Distribusi ini adalah kasus khusus Gamma.

6.12 4.11 Hubungan Antardistribusi

Bagian ini penting karena membantu mahasiswa melihat probabilitas sebagai jaringan ide, bukan daftar rumus terpisah.

6.12.1 Exponential ↔︎ Poisson Process

Jika banyak kejadian dalam interval mengikuti Poisson process dengan rate \(\lambda\), maka waktu antar kedatangan mengikuti Exponential(\(\lambda\)).

6.12.2 Erlang ↔︎ Exponential

Erlang adalah jumlah beberapa random variable Exponential independen dengan rate sama.

6.12.3 Gamma ↔︎ Erlang

Erlang adalah kasus khusus Gamma dengan parameter shape berupa bilangan bulat positif.

6.12.4 Normal ↔︎ Binomial

Binomial dengan \(n\) besar sering dapat didekati oleh Normal.

6.12.5 Chi-Square ↔︎ Gamma

Chi-Square adalah kasus khusus Gamma.

6.17 4.16 Kesalahan Umum yang Harus Dihindari

  1. Menganggap tinggi PDF adalah peluang
    Tidak. Peluang adalah area di bawah kurva.

  2. Menghitung \(P(X=x)\) untuk kontinu seolah-olah positif
    Dalam model kontinu, peluang titik tunggal adalah nol.

  3. Memakai Exponential untuk semua lifetime
    Exponential hanya cocok bila hazard rate konstan / memoryless.

  4. Memakai Normal untuk data yang sangat skewed atau bernilai negatif-padahal tidak mungkin
    Pilih model sesuai konteks.

  5. Menghafal distribusi tanpa memahami makna parameter
    Yang penting bukan sekadar nama distribusinya, tetapi arti shape, scale, mean, variance, dan implikasinya.

6.18 4.17 Menyimpulkan Bab Ini

Distribusi kontinu memberi kita bahasa untuk memodelkan banyak besaran dunia nyata yang berupa pengukuran, waktu, umur hidup, dan nilai-nilai positif yang berubah halus.

Kita telah melihat: - uniform kontinu untuk interval yang merata, - normal untuk fenomena simetris dan agregatif, - gamma untuk waktu tunggu positif yang fleksibel, - exponential untuk memoryless waiting time, - erlang untuk jumlah beberapa exponential, - weibull untuk reliability yang lebih fleksibel, - pareto untuk heavy-tail, - chi-square untuk inferensi statistik dan hubungan dengan normal.

Di samping itu, hubungan antardistribusi membantu kita melihat probabilitas sebagai sistem ide yang saling terhubung.

6.19 4.18 Ringkasan Poin Inti

  1. Untuk random variable kontinu, peluang diperoleh dari area di bawah PDF.
  2. Distribusi kontinu kustom bisa dibangun dari tabel atau interpolasi numerik.
  3. Uniform kontinu cocok untuk interval merata.
  4. Normal penting untuk banyak fenomena simetris dan sebagai pendekatan Binomial.
  5. Exponential memodelkan waiting time memoryless dan terkait erat dengan Poisson process.
  6. Gamma dan Erlang memodelkan akumulasi waiting time.
  7. Weibull sangat penting dalam reliability engineering.
  8. Pareto penting untuk fenomena ekstrem dan heavy-tail.
  9. Chi-Square adalah distribusi penting dalam statistika inferensial.
  10. Memahami hubungan antardistribusi membantu membangun intuisi yang lebih dalam.

1. Komponen Mesin Fotokopi (Distribusi Eksponensial) Fungsi probabilitas: \(f(x) = \frac{e^{-x/1000}}{1000}\) untuk \(x > 0\). * a. Bertahan > 3000 jam: \(P(X > 3000) = 1 - P(X \le 3000) = e^{-3000/1000} = e^{-3} \approx \mathbf{0,049787}\). * b. Gagal antara 1000-2000 jam: \(P(1000 < X < 2000) = F(2000) - F(1000) = (1 - e^{-2}) - (1 - e^{-1}) \approx \mathbf{0,232544}\). * c. Gagal < 1000 jam: \(P(X < 1000) = 1 - e^{-1} \approx \mathbf{0,63212}\). * d. Waktu saat 10% gagal: \(P(X < x) = 0,1 \implies 1 - e^{-x/1000} = 0,1 \implies e^{-x/1000} = 0,9 \implies x = -1000 \ln(0,9) \approx \mathbf{105,36 \text{ jam}}\).

2. Lebar Gap Kepala Rekaman Magnetik (Distribusi Kontinu Acak) Fungsi: \(f(x) = 0,5x\) untuk \(0 < x < 2\). * Fungsi Distribusi Kumulatif (CDF): Diperoleh dari integral \(f(x)\): \(F(x) = 0\) untuk \(x \le 0\); \(F(x) = \mathbf{0,25x^2}\) untuk \(0 < x < 2\); \(F(x) = 1\) untuk \(x \ge 2\).

3. Ketebalan Conductive Coating Fungsi: \(f(x) = 600x^{-2}\) untuk \(100 < x < 120\). * a. Rata-rata & Variansi: \(\mu = E(X) = \int_{100}^{120} x(600x^{-2}) dx = \int_{100}^{120} \frac{600}{x} dx = 600 \ln(1.2) \approx \mathbf{109,39}\). \(\sigma^2 = \int_{100}^{120} (x - 109,39)^2 600x^{-2} dx \approx \mathbf{33,186}\). * b. 4. Partikel Kontaminasi Fungsi: \(f(x) = 2x^{-3}\) untuk \(x > 1\). * Rata-rata: \(\mu = \int_{1}^{\infty} x(2x^{-3}) dx = \int_{1}^{\infty} 2x^{-2} dx = [-2x^{-1}]_1^\infty = \mathbf{2}\).

5. Senyawa Kimia (Distribusi Seragam) Interval: \(49,75 < x < 50,25\). * a. Rata-rata & Variansi: \(\mu = \frac{49,75 + 50,25}{2} = \mathbf{50}\). Variansi \(\sigma^2 = \frac{(50,25 - 49,75)^2}{12} \approx \mathbf{0,02083}\). * b. CDF: \(F(x) = \frac{x - 49,75}{50,25 - 49,75} = \mathbf{2(x - 49,75)}\). * c. \(P(X < 50,1)\): \(F(50,1) = 2(50,1 - 49,75) = \mathbf{0,7}\). * d. \(P(49,9 < X < 50)\): \(F(50) - F(49,9) = 0,5 - 0,3 = \mathbf{0,2}\).

Jika \(P(X > x) = 0,1 \implies \mathbf{x = 0,214}\). * d. Rata-rata & Variansi: \(\mu = \mathbf{0,210}\), \(\sigma^2 = \frac{(0,215 - 0,205)^2}{12} = \mathbf{8,333 \times 10^{-6}}\).

11. Regulator Voltase (Distribusi Eksponensial) \(\beta = 6\) tahun. * a. Peluang gagal: Tidak memiliki memori, \(P(X < 6) = 1 - e^{-6/6} = \mathbf{0,63212}\). * b. Prakiraan gagal selanjutnya: Karena memoryless, rata-rata waktunya tetap \(\mathbf{6 \text{ tahun}}\).

Kegagalan kedua: \(E(X_1 + X_2) = 25000 + 25000 = \mathbf{50.000 \text{ jam}}\). * c. Kegagalan ketiga > 50.000 jam (Erlang): \(P(X_1+X_2+X_3 > 50000) = 1 - 0,323324 = \mathbf{0,676676}\).

Rata-rata bit ke 5 error: \(\frac{r}{\lambda} = \frac{5}{10^{-5}} = \mathbf{5 \times 10^5 \text{ bits}}\). * b. Standar deviasi: \(\sqrt{r/\lambda^2} = \sqrt{5 \times 10^{10}} \text{ bits}\). 6.22.2 Bagian 2: Solusi Kode Python (Berbasis Class)

Kode di bawah ini membungkus penyelesaian semua soal probabilitas kontinu di atas menjadi class ContinuousDistributions yang dapat di-reuse.

import scipy.stats as stats
import scipy.integrate as integrate
import math

class ContinuousDistributions:
    """Class toolkit untuk menyelesaikan Problem Set 5 Probabilitas & Statistik"""
    
    @staticmethod
    def exponential_prob(mean, x, prob_type="less_than"):
        """Distribusi Eksponensial (Soal 1, 11, 12, 17)"""
        if prob_type == "less_than":
            return stats.expon.cdf(x, scale=mean)
        return stats.expon.sf(x, scale=mean)

    @staticmethod
    def uniform_stats(a, b):
        """Mean dan Variansi Distribusi Seragam (Soal 5, 6)"""
        return (a + b) / 2, ((b - a)**2) / 12

    @staticmethod
    def uniform_prob(a, b, x, prob_type="less_than"):
        """Distribusi Seragam Kontinu (Soal 5, 6)"""
        if prob_type == "less_than":
            return stats.uniform.cdf(x, loc=a, scale=b-a)
        return stats.uniform.sf(x, loc=a, scale=b-a)

    @staticmethod
    def custom_pdf_stats(pdf_func, a, b):
        """Kalkulasi mean & variansi untuk PDF kustom via integrasi (Soal 3, 4)"""
        mean, _ = integrate.quad(lambda x: x * pdf_func(x), a, b)
        var_raw, _ = integrate.quad(lambda x: (x**2) * pdf_func(x), a, b)
        return mean, var_raw - (mean**2)

    @staticmethod
    def normal_prob(mu, sigma, x, prob_type="less_than"):
        """Distribusi Normal (Soal 7, 8, 9, 10)"""
        if prob_type == "less_than":
            return stats.norm.cdf(x, loc=mu, scale=sigma)
        return stats.norm.sf(x, loc=mu, scale=sigma)

    @staticmethod
    def normal_inverse(mu, sigma, p, prob_type="less_than"):
        """Pencarian batas X pada probabilitas Normal tertentu (Soal 7c, 8c)"""
        if prob_type == "less_than":
            return stats.norm.ppf(p, loc=mu, scale=sigma)
        return stats.norm.isf(p, loc=mu, scale=sigma)

    @staticmethod
    def erlang_prob(r, lam, x, prob_type="greater_than"):
        """Distribusi Erlang/Gamma untuk r kejadian (Soal 13c)"""
        scale = 1 / lam
        if prob_type == "greater_than":
            return stats.erlang.sf(x, r, scale=scale)
        return stats.erlang.cdf(x, r, scale=scale)

    @staticmethod
    def weibull_prob(alpha, beta, x, prob_type="less_than"):
        """Distribusi Weibull (Soal 15)"""
        # Konversi scipy weibull parameters
        scale = alpha ** (-1 / beta)
        if prob_type == "less_than":
            return stats.weibull_min.cdf(x, c=beta, scale=scale)
        return stats.weibull_min.sf(x, c=beta, scale=scale)

    @staticmethod
    def poisson_prob(mu, x, prob_type="greater_equal"):
        """Distribusi Poisson diskrit standar (Soal 14c)"""
        if prob_type == "greater_equal":
            return stats.poisson.sf(x-1, mu)
        return stats.poisson.cdf(x, mu)
