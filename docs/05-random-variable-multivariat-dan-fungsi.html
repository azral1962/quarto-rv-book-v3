7.1 Pembuka

Sampai bab sebelumnya, kita sering berbicara seolah-olah dunia hanya memiliki satu random variable pada satu waktu: - jumlah cacat, - waktu tunggu, - umur hidup, - profit, - jumlah pelanggan.

Namun kenyataan jauh lebih kaya. Dalam dunia nyata, keputusan hampir selalu melibatkan lebih dari satu peubah acak.

Contoh: - tinggi dan berat badan, - permintaan dan pendapatan, - jumlah pelanggan dan waktu layanan, - suhu dan konsumsi listrik, - arus dan daya, - umur komponen dan biaya klaim.

Begitu ada lebih dari satu random variable, muncul pertanyaan-pertanyaan baru: - apakah mereka saling memengaruhi? - apakah keduanya independen? - kalau satu naik, apakah yang lain cenderung naik? - bagaimana distribusi salah satunya bila yang lain diketahui? - bagaimana distribusi output jika output adalah fungsi dari input yang acak?

Bab ini menjawab pertanyaan-pertanyaan itu.

7.2 5.1 Quick Win: Dua Peubah, Satu Plot

Mari mulai dengan simulasi dua random variable yang berkorelasi positif. Misalnya: - \(X\) = jam belajar, - \(Y\) = skor kuis.

Ini tentu penyederhanaan, tetapi cukup baik untuk melihat ide dasar.

import numpy as np
import matplotlib.pyplot as plt
from scipy import stats

rng = np.random.default_rng(42)

mean = [5, 70]
cov = [[1.5, 8],
       [8, 80]]

data = rng.multivariate_normal(mean, cov, size=1000)
X = data[:, 0]
Y = data[:, 1]

plt.figure(figsize=(6,5))
plt.scatter(X, Y, alpha=0.4, s=15)
plt.xlabel("Jam belajar")
plt.ylabel("Skor kuis")
plt.title("Dua random variable dengan kecenderungan positif")
plt.grid(alpha=0.3)
plt.show()
print("Mean X =", X.mean())
print("Mean Y =", Y.mean())
print("Cov(X,Y) =", np.cov(X, Y, ddof=0)[0,1])
print("Corr(X,Y) =", np.corrcoef(X, Y)[0,1])
Mean X = 5.0966118178007385
Mean Y = 70.63531660231106
Cov(X,Y) = 8.268878410266762
Corr(X,Y) = 0.7409518158857897

Dari quick win ini, kita langsung melihat dua hal: 1. dunia nyata sering lebih masuk akal bila dimodelkan dengan lebih dari satu peubah acak, 2. hubungan antar peubah dapat diringkas dengan konsep seperti covariance dan correlation.

7.3 5.2 Joint Distribution: Distribusi Bersama

7.3.1 K — Konteks

Misalkan: - \(X\) = jumlah pelanggan datang per jam, - \(Y\) = total pendapatan per jam.

Atau: - \(X\) = jumlah produk cacat dari supplier A, - \(Y\) = jumlah produk cacat dari supplier B.

Kita tidak cukup hanya tahu distribusi \(X\) sendiri atau \(Y\) sendiri. Kita ingin tahu bagaimana keduanya muncul bersama.

7.3.2 M — Model

Distribusi bersama atau joint distribution mendeskripsikan peluang dua random variable secara simultan.

7.3.2.1 Untuk diskrit

\[ p_{X,Y}(x,y)=P(X=x, Y=y) \]

7.3.2.2 Untuk kontinu

\[ f_{X,Y}(x,y) \]

Joint distribution memberi informasi lengkap tentang pasangan \((X,Y)\).

7.3.3 Q — Questions

  1. Berapa peluang \(X\) dan \(Y\) mengambil nilai tertentu secara bersamaan?
  2. Apakah nilai besar pada \(X\) sering disertai nilai besar pada \(Y\)?
  3. Bagaimana menurunkan distribusi salah satu peubah dari distribusi bersama?

7.3.4 A — Apply (diskrit sederhana)

Misalkan dua koin fair dilempar. Definisikan: - \(X\) = jumlah Head, - \(Y\) = indikator apakah koin pertama Head (1) atau tidak (0).

Mari bangun joint distribution-nya.

import itertools

outcomes = list(itertools.product(['H', 'T'], repeat=2))

def X_fun(o):
    return o.count('H')

def Y_fun(o):
    return 1 if o[0] == 'H' else 0

pairs = [(X_fun(o), Y_fun(o)) for o in outcomes]
pairs
[(2, 1), (1, 1), (1, 0), (0, 0)]
from collections import Counter

counter = Counter(pairs)
joint = {k: v/len(outcomes) for k,v in counter.items()}
joint
{(2, 1): 0.25, (1, 1): 0.25, (1, 0): 0.25, (0, 0): 0.25}

Joint distribution ini memberi peluang bersama, misalnya: \[ P(X=2, Y=1) \]

yang berarti: - jumlah Head = 2, - dan koin pertama = Head.

\(x \backslash y\) | 0 | 1 |

7.5 5.4 Marginal Distribution

7.5.1 K — Konteks

Kadang kita hanya ingin distribusi satu peubah, tanpa memperhatikan peubah lainnya.

7.5.2 M — Model

Dari joint distribution, kita bisa memperoleh distribusi marginal.

7.5.2.1 Untuk diskrit

\[ p_X(x)=\sum_y p_{X,Y}(x,y) \]

\[ p_Y(y)=\sum_x p_{X,Y}(x,y) \]

7.5.3 Q — Questions

  1. Bagaimana “mengabaikan” peubah lain secara formal?
  2. Apa arti marginal dalam keputusan?

7.5.4 A — Apply

pX = joint_table.sum(axis=1)  ## jumlahkan sepanjang kolom
pY = joint_table.sum(axis=0)  ## jumlahkan sepanjang baris

print("Marginal pX =", pX)
print("Marginal pY =", pY)
Marginal pX = [0.3 0.4 0.3]
Marginal pY = [0.35 0.65]
plt.figure(figsize=(8,3))

plt.subplot(1,2,1)
plt.bar([0,1,2], pX)
plt.title("Marginal pX")
plt.xlabel("x")
plt.ylabel("P(X=x)")
plt.grid(alpha=0.3)

plt.subplot(1,2,2)
plt.bar([0,1], pY)
plt.title("Marginal pY")
plt.xlabel("y")
plt.ylabel("P(Y=y)")
plt.grid(alpha=0.3)

plt.tight_layout()
plt.show()

7.5.4.1 Interpretasi

Marginal distribution memberi gambaran perilaku satu peubah ketika peubah lain tidak diperhatikan secara eksplisit.

7.6 5.5 Conditional Distribution

7.6.1 K — Konteks

Dalam keputusan nyata, sering kali kita sudah mengetahui sebagian informasi. Misalnya: - distribusi pendapatan per jam dengan syarat jumlah pelanggan = 10, - distribusi nilai ujian dengan syarat jam belajar > 5, - distribusi demand dengan syarat hari hujan.

7.6.2 M — Model

7.6.2.1 Untuk diskrit

Jika \(P(Y=y)>0\), maka:

\[ P(X=x \mid Y=y)=\frac{P(X=x, Y=y)}{P(Y=y)} \]

7.6.3 Q — Questions

  1. Bagaimana distribusi satu peubah berubah jika yang lain diketahui?
  2. Mengapa conditional distribution penting untuk inferensi dan keputusan?

7.6.4 A — Apply

Hitung \(P(X=x \mid Y=1)\) dari tabel joint sebelumnya.

## Conditional p(X|Y=1)
cond_X_given_Y1 = joint_table[:, 1] / pY[1]
print("P(X=x | Y=1) =", cond_X_given_Y1)
print("Jumlah =", cond_X_given_Y1.sum())
P(X=x | Y=1) = [0.30769231 0.38461538 0.30769231]
Jumlah = 1.0
plt.figure(figsize=(6,4))
plt.bar([0,1,2], cond_X_given_Y1)
plt.xlabel("x")
plt.ylabel("P(X=x | Y=1)")
plt.title("Conditional distribution X | Y=1")
plt.grid(alpha=0.3)
plt.show()

7.6.4.1 Interpretasi

Conditional distribution menunjukkan bahwa ketika informasi tambahan tersedia, distribusi yang relevan untuk keputusan juga berubah.

7.7 5.6 Independensi

7.7.1 K — Konteks

Kadang dua peubah tampak tidak saling memengaruhi. Kadang justru sangat terkait. Membedakan dua situasi ini penting.

7.7.2 M — Model

Dua random variable \(X\) dan \(Y\) dikatakan independen jika:

7.7.2.1 Untuk diskrit

\[ P(X=x, Y=y)=P(X=x)P(Y=y) \]

untuk semua \(x,y\).

7.7.2.2 Untuk kontinu

\[ f_{X,Y}(x,y)=f_X(x)f_Y(y) \]

7.7.3 Q — Questions

  1. Bagaimana menguji independensi dari joint distribution?
  2. Apakah korelasi nol berarti pasti independen?
  3. Kapan independensi terlalu kuat sebagai asumsi?

7.7.4 A — Apply

Mari cek apakah joint_table tadi independen.

product_table = np.outer(pX, pY)

print("Joint table:\n", joint_table)
print("\nProduct of marginals:\n", product_table)
print("\nApakah sama? ", np.allclose(joint_table, product_table))
Joint table:
 [[0.1  0.2 ]
 [0.15 0.25]
 [0.1  0.2 ]]

Product of marginals:
 [[0.105 0.195]
 [0.14  0.26 ]
 [0.105 0.195]]

Apakah sama?  False

7.7.4.1 Interpretasi

Jika joint distribution tidak sama dengan hasil kali marginals, maka \(X\) dan \(Y\) tidak independen.

7.7.5 Catatan penting

  • Independen ⇒ korelasi nol (dalam banyak kasus dengan momen terbatas),
  • tetapi korelasi nol tidak selalu berarti independen.

Nanti kita lihat contoh kecilnya.

7.8 5.7 Bivariat Kontinu

7.8.1 K — Konteks

Banyak pasangan peubah acak lebih alami dimodelkan kontinu, misalnya: - tinggi dan berat badan, - suhu dan konsumsi listrik, - waktu proses tahap 1 dan tahap 2.

7.8.2 M — Model

Joint density \(f_{X,Y}(x,y)\) memberi kerapatan peluang pasangan \((X,Y)\).

Peluang pada daerah \(A\) adalah: \[ P((X,Y)\in A)=\iint_A f_{X,Y}(x,y)\,dx\,dy \]

7.8.3 Q — Questions

  1. Bagaimana membayangkan distribusi bersama kontinu?
  2. Bagaimana memperoleh marginal?
  3. Bagaimana memvisualisasikannya?

7.8.4 A — Apply

Mari ambil contoh Normal bivariat.

mean = [0, 0]
cov = [[1, 0.8],
       [0.8, 1]]

data = rng.multivariate_normal(mean, cov, size=5000)
X = data[:,0]
Y = data[:,1]

plt.figure(figsize=(6,5))
plt.scatter(X, Y, alpha=0.2, s=8)
plt.xlabel("X")
plt.ylabel("Y")
plt.title("Scatterplot Normal bivariat")
plt.grid(alpha=0.3)
plt.show()

Marginalnya dapat dilihat dari histogram tiap komponen.

plt.figure(figsize=(8,3))

plt.subplot(1,2,1)
plt.hist(X, bins=40, density=True, alpha=0.7)
plt.title("Marginal X")
plt.grid(alpha=0.3)

plt.hist(Y, bins=40, density=True, alpha=0.7)
plt.title("Marginal Y")
plt.grid(alpha=0.3)

7.9 5.8 Kovariansi

7.9.1 K — Konteks

Kita sering ingin tahu apakah dua peubah cenderung bergerak bersama. Misalnya: - semakin banyak pelanggan, semakin besar pendapatan, - semakin lama hujan, semakin kecil pengunjung, - semakin besar arus, semakin besar daya.

7.9.2 M — Model

Kovariansi didefinisikan sebagai:

\[ \mathrm{Cov}(X,Y)=E[(X-E[X])(Y-E[Y])] \]

Bentuk ekuivalennya: \[ \mathrm{Cov}(X,Y)=E[XY]-E[X]E[Y] \]

7.9.3 Q — Questions

  1. Apa arti kovariansi positif, negatif, atau nol?
  2. Mengapa satuan kovariansi sulit ditafsirkan langsung?
  3. Bagaimana menghitungnya dari data?

7.9.4 A — Apply

cov_xy = np.cov(X, Y, ddof=0)[0,1]
print("Cov(X,Y) =", cov_xy)
Cov(X,Y) = 0.7693398942915717

7.9.4.1 Interpretasi

  • Kovariansi positif: saat \(X\) di atas rata-rata, \(Y\) cenderung juga di atas rata-rata.
  • Kovariansi negatif: saat \(X\) di atas rata-rata, \(Y\) cenderung di bawah rata-rata.
  • Kovariansi nol: tidak ada hubungan linear yang jelas, tetapi belum tentu independen.

7.10 5.9 Korelasi

7.10.1 K — Konteks

Kovariansi bagus, tetapi nilainya dipengaruhi satuan. Kita sering butuh ukuran yang lebih mudah dibandingkan antar-kasus.

7.10.2 M — Model

Korelasi adalah versi ternormalisasi dari kovariansi:

\[ \rho_{X,Y} = \frac{\mathrm{Cov}(X,Y)}{\sigma_X \sigma_Y} \]

Nilainya berada antara -1 dan 1.

7.10.3 Q — Questions

  1. Apa arti korelasi mendekati 1, 0, atau -1?
  2. Apa bedanya korelasi dan kovariansi?
  3. Mengapa korelasi tidak otomatis berarti hubungan sebab-akibat?

7.10.4 A — Apply

corr_xy = np.corrcoef(X, Y)[0,1]
print("Corr(X,Y) =", corr_xy)
Corr(X,Y) = 0.7855402048546878

7.10.4.1 Interpretasi

  • \(\rho \approx 1\): hubungan linear positif kuat,
  • \(\rho \approx -1\): hubungan linear negatif kuat,
  • \(\rho \approx 0\): hubungan linear lemah atau tidak ada.

7.10.5 Peringatan penting

Korelasi: - tidak berarti sebab-akibat, - tidak menangkap semua hubungan nonlinear, - bisa nol meskipun dua peubah sangat bergantung secara nonlinear.

7.11 5.10 Contoh: Korelasi Nol tetapi Tidak Independen

Ambil: - \(X \sim \mathrm{Uniform}(-1,1)\) - \(Y = X^2\)

Maka jelas \(Y\) ditentukan sepenuhnya oleh \(X\), jadi mereka tidak independen. Namun korelasinya bisa nol.

rng = np.random.default_rng(123)
X = rng.uniform(-1, 1, size=100000)
Y = X**2

print("Corr(X,Y) ~", np.corrcoef(X, Y)[0,1])
Corr(X,Y) ~ -0.0005791433843902291
plt.figure(figsize=(6,5))
plt.scatter(X, Y, alpha=0.15, s=8)
plt.xlabel("X")
plt.ylabel("Y = X^2")
plt.title("Korelasi hampir nol, tetapi jelas tidak independen")
plt.grid(alpha=0.3)
plt.show()

7.11.0.1 Pelajaran penting

Korelasi nol bukan bukti independensi.

7.12 5.11 Fungsi dari Random Variable

7.12.1 K — Konteks

Sering kali yang kita minati bukan random variable awal, tetapi fungsi darinya.

Contoh: - jika arus \(I\) acak, maka daya \(P = I^2R\), - jika harga \(H\) dan jumlah terjual \(Q\) acak, maka revenue \(R = HQ\), - jika dua tahap layanan acak, total waktu \(T = X_1 + X_2\).

7.12.2 M — Model

Jika \(X\) adalah random variable dan \(Y=g(X)\), maka \(Y\) juga random variable.

Untuk dua peubah, jika: \[ Z = g(X,Y) \] maka \(Z\) juga random variable.

7.12.3 Q — Questions

  1. Bagaimana distribusi berubah setelah transformasi?
  2. Mengapa fungsi nonlinear bisa mengubah bentuk distribusi secara drastis?
  3. Bagaimana Python membantu melihat perubahan ini?

7.12.4 A — Apply

Misalkan: \[ I \sim \mathcal{N}(10, 0.8^2), \qquad P = I^2R, \quad R=5 \]

rng = np.random.default_rng(7)
I = rng.normal(loc=10, scale=0.8, size=100000)
R = 5
P = (I**2) * R

print("Mean I =", I.mean(), "Var I =", I.var())
print("Mean P =", P.mean(), "Var P =", P.var())
Mean I = 9.998938944730092 Var I = 0.6378216295132455
Mean P = 503.0830082497668 Var P = 6392.509470652347
plt.figure(figsize=(8,4))
plt.hist(I, bins=60, density=True, alpha=0.6, label="I")
plt.hist(P, bins=60, density=True, alpha=0.6, label="P = I^2 R")
plt.xlabel("Nilai")
plt.ylabel("Density")
plt.title("Fungsi nonlinear mengubah bentuk distribusi")
plt.legend()
plt.grid(alpha=0.3)
plt.show()

7.12.4.1 Interpretasi

Transformasi nonlinear dapat: - mengubah pusat, - mengubah varians, - mengubah skewness, - mengubah peluang melewati batas aman.

7.13 5.12 Fungsi dari Dua Random Variable

7.13.1 K — Konteks

Banyak keputusan melibatkan gabungan dua random variable.

Contoh: - revenue \(R = P \times Q\), - total waktu \(T = X + Y\), - keuntungan \(G = \text{revenue} - \text{cost}\).

7.13.2 M — Model

Jika \(X\) dan \(Y\) acak, maka: - \(X+Y\), - \(XY\), - \(aX+bY\), - \(\max(X,Y)\), - dan fungsi lain dari keduanya juga random variable.

7.13.3 Q — Questions

  1. Bagaimana rata-rata jumlah dua random variable?
  2. Bagaimana varians jumlah dua random variable?
  3. Kapan covariance ikut berperan?

7.13.4 A — Apply

Untuk dua random variable: \[ E[X+Y] = E[X] + E[Y] \]

dan: \[ \mathrm{Var}(X+Y)=\mathrm{Var}(X)+\mathrm{Var}(Y)+2\mathrm{Cov}(X,Y) \]

Mari verifikasi dengan simulasi.

rng = np.random.default_rng(11)

       [3, 9]]

Y = data[:,1]
T = X + Y

lhs_mean = T.mean()
rhs_mean = X.mean() + Y.mean()

lhs_var = T.var()
rhs_var = X.var() + Y.var() + 2*np.cov(X, Y, ddof=0)[0,1]

print("Mean right =", rhs_mean)

Jika dua peubah berkorelasi positif, variance jumlahnya lebih besar daripada jika mereka independen. Ini sangat penting dalam: - risk aggregation, - portfolio, - total demand, - total delay.

7.25.1 Kategori 1: Distribusi Multinomial (Diskrit Gabungan)

Prinsip yang Digunakan: Distribusi Multinomial adalah perluasan dari distribusi Binomial. Distribusi ini digunakan ketika sebuah eksperimen memiliki lebih dari dua kemungkinan hasil (mutually exclusive) pada setiap percobaannya, dan percobaan tersebut dilakukan sebanyak \(n\) kali secara independen. Soal & Solusi Representatif: * Kasus Oven Cacat: 4 oven jatuh. Standar deviasi \(\approx 0,1225\). * P(D < 5,9): \(P(Z < \frac{5,9 - 6}{0,1225}) = P(Z < -0.816) \approx \mathbf{0,207}\). * Kasus Portofolio Investasi: \(X_1\) (Keamanan 1): \(\mu=5\%, \sigma=2\%\). \(X_2\) (Keamanan 2): \(\mu=5\%, \sigma=4\%\). Investasi dibagi rata 50:50. * \(R = 0,5 X_1 + 0,5 X_2\). * Rata-rata Return: \(0,5(5) + 0,5(5) = \mathbf{5\%}\). * Variansi dengan korelasi: \(\sigma_R^2 = a^2\sigma_1^2 + b^2\sigma_2^2 + 2ab\cdot Cov(X_1,X_2)\) di mana \(Cov = \rho \sigma_1 \sigma_2\). \(\sigma_R^2 = (0,5^2)(2^2) + (0,5^2)(4^2) + 2(0,5)(0,5)(-0,5)(2)(4) = 1 + 4 - 2 = \mathbf{3}\). * Standar deviasi: \(\mathbf{\sqrt{3} \approx 1,732\%}\). Nilai ini lebih rendah dibanding investasi hanya pada keamanan pertama (2%), menunjukkan prinsip diversifikasi meminimalkan risiko.

7.25.4 Kategori 4: Kovariansi, Korelasi, dan Normal Bivariat

Prinsip yang Digunakan: Kovariansi mengukur hubungan linear antara dua variabel, dan Korelasi (\(\rho\)) menyelaraskan skala kovariansi ke rentang \(-1 \le \rho \le 1\). Jika \(\rho=0\) pada distribusi normal bivariat, maka kedua variabel dipastikan independen.

Soal & Solusi Representatif: * Kasus Spesifikasi Tinta Luminesen: \(X \sim N(0,1; 0,00031^2)\), \(Y \sim N(0,23; 0,00017^2)\), \(\rho=0\). Spesifikasi: \(0,099535 < X < 0,100465\) dan \(0,22966 < Y < 0,23034\). * Karena \(\rho=0\), kejadian \(X\) dan \(Y\) independen. Kita bisa menghitung probabilitas total sebagai \(P(Specs X) \times P(Specs Y)\). * \(Z_{X1} = \frac{0,099535 - 0,1}{0,00031} = -1,5\) dan \(Z_{X2} = 1,5\). Area \(P(-1,5 < Z < 1,5) \approx 0,8664\). * \(Z_{Y1} = \frac{0,22966 - 0,23}{0,00017} = -2\) dan \(Z_{Y2} = 2\). Area \(P(-2 < Z < 2) \approx 0,9545\). * Probabilitas lampu lulus uji: \(0,8664 \times 0,9545 \approx \mathbf{0,827}\).

7.25.5 Desain Kelas Python Multivariat (Reusable)

Skrip di bawah ini dirancang menggunakan scipy.integrate dan scipy.stats untuk menjawab setiap keluarga masalah di atas.

import numpy as np
import scipy.stats as stats
import scipy.integrate as integrate
import math

class MultivariateSolver:
    
    @staticmethod
    def multinomial_prob(n, x_list, p_list):
        """
        Menghitung Probabilitas Distribusi Multinomial (Kasus Oven Cacat, Obat)
        x_list: list jumlah observasi untuk tiap kategori
        p_list: list probabilitas untuk tiap kategori
        """
        prob = math.factorial(n)
        for x in x_list:
            prob /= math.factorial(x)
        for x, p in zip(x_list, p_list):
            prob *= (p ** x)
        return prob
    
    @staticmethod
    def find_continuous_constant_2D(f_xy, x_bounds, y_bounds):
        """
        Mencari konstanta 'c' agar integral f(x,y) = 1
        x_bounds: list [min, max], y_bounds: list [min, max] atau fungsi lambda
        """
        if callable(y_bounds): # Jika batas Y bergantung pada X
            integral_val, _ = integrate.dblquad(f_xy, x_bounds, x_bounds, y_bounds, y_bounds)
        else:
            integral_val, _ = integrate.dblquad(f_xy, x_bounds, x_bounds, lambda x: y_bounds, lambda x: y_bounds)
        return 1 / integral_val

    @staticmethod
    def linear_combination_normal(means, std_devs, weights):
        """
        Menghitung distribusi Y = a1*X1 + a2*X2 ... dari variabel Normal Independen
        (Kasus Casing Plastik, Komponen U-Shape)
        """
        mu_Y = sum(w * m for w, m in zip(weights, means))
        var_Y = sum((w**2) * (s**2) for w, s in zip(weights, std_devs))
        return mu_Y, math.sqrt(var_Y)

    @staticmethod
    def portfolio_risk(w1, w2, mu1, mu2, std1, std2, corr):
        """
        Kalkulasi E(R) dan Var(R) untuk kombinasi Linear Dependent (Kasus Portofolio Investasi)
        """
        exp_return = w1*mu1 + w2*mu2
        covariance = corr * std1 * std2
        variance = (w1**2)*(std1**2) + (w2**2)*(std2**2) + 2*w1*w2*covariance
        return exp_return, math.sqrt(variance)
